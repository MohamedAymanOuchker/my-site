# Mohamed Ayman Ouchker - Personal Portfolio

A modern, interactive 3D portfolio website built to showcase robotics, AI, and software engineering projects.

## 🚀 Technologies Used

- **Frontend Framework:** React 18 with Vite
- **Styling:** Tailwind CSS
- **3D Graphics:** React Three Fiber & Three.js
- **Animations:** Framer Motion
- **Smooth Scrolling:** Lenis
- **Language:** TypeScript

## 🌟 Featured Projects

- **Educational Robot:** Affordable educational robotics platform for teaching programming to children.
- **RoboSense:** Open-source telemetry layer for robot and IoT sensor readings.
- **AgriBot:** Autonomous agricultural robot using computer vision.
- **AI-Powered Face Detection:** Real-time emotion recognition system.
- **DronePy:** Drone Programming With Python - A platform for controlling and programming drones autonomously using Python.
- *And more inside the portfolio...*

## 🛠️ Getting Started

### Prerequisites

- Node.js (v18+ recommended)
- npm or yarn

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```
